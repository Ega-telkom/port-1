# Keterampilan

## Geospasial

(C) Kontributor OpenStreetMap, Petak Transportasi milik Andy Alan


Sebagai Eganatha, Saya suka membuat fitur Transportasi Publik, proposal pemerintah, guna-lahan, dan masih banyak lagi dalam OpenStreetMap. Saya mahir dalam pengunaan format geodata terbuka, contohnya, geojson, gpx, shapefile, osm, dan program maupun layanan web yang membuat kontribusi dalam OSM dapat berjalan dengan lancar, JOSM dan API Overpass.

## Foto/Sinemato-grafi

Buatan Sendiri: Gambar eksposur panjang, aku beri nama "noda foton"


Saya mengetahui sedikit pengetahuan tentang fotografi dan sinematografi. Sebelumnya saya sama sekali tidak tertarik dengan pemrograman, waktu telah berubah dan ini menjadi hal kedua yang saya lakukan untuk berjaga-jaga.

## Bahasa Pemrograman C

C adalah bahasa pemrograman tingkat rendah. Saya akhirnya menyukai C karena saya terpengaruh oleh komunitas Suckless, dan juga karena cara kerjanya yang menyediakan akses rendah ke memori! Setelah itu saya mulai mengkompilasi banyak program (sebagian besar ditulis dalam bahasa C) dan memodifikasinya sambil belajar bahasa C di sepanjang jalan.

## Markup & pengaturan-huruf

Elementi di Dinamica e simulazione di volo - Agostino De Marco & Domenico P. Coiro


Saya menggunakan LaTeX untuk sebagian besar esai dan laporan saya, karena saya dapat mengetik apa pun yang saya inginkan tanpa perlu repot-repot memikirkan gaya dokumen dan sebagainya. HTML juga digunakan untuk sebagian besar halaman web saya dan disertai dengan beberapa stylesheet eksternal, dan alat bantu pembuatan.

### HTML5/CSS

Saya belajar HTML dan CSS ketika saya bermain-main dengan stylesheet situs web dan membuatnya sesuai dengan yang saya inginkan. Segera saya mempelajari standar HTML5 dan CSS dasar, saya tidak ingin terlalu banyak belajar dalam keterampilan ini, tetapi cukup untuk memudahkan proyek-proyek saya yang lain.

## Semi audiofil

Saya mendengarkan musik seperti yang dilakukan orang lain, tetapi terkadang yang memiliki ketepatan tinggi. Ketepatan tinggi tidak selalu berarti kualitas tinggi, ini berarti salinan lagu tersebut harus terdengar sedekat mungkin dengan rekaman di studio. Jangan tanya saya dari mana saya mendapatkan lagu-lagu itu...

## Lainnya

Saya membiasakan diri saya dengan menggunakan sistem mirip-UNIX sejak lama, seperti GNU/Linux. Menggunakan sistem tersebut membuat perkembangan saya jauh lebih mudah berkat sifat Open Source. Secara spesifik, saya menggunakan Fedora Workstation 41 di laptop saya, dan Arch Linux yang menjalankan KDE Plasma 6.3 di desktop di rumah. Saat ini saya jarang menggunakan MicrosoftÂ® Windows.
