# port-1
Sebuah portfolio versi 1 milik Ega.

## Memulai & Membangun
Untuk memulai server:
```sh
npm run start
```

Untuk membangun `tailwindcss` secara otomatis:
```sh
npm run build:css
```

Untuk membangun `pages/**/*.html` secara manual:
```sh
npm run build
```

Untuk membangun `pages/**/*.html` secara otomatis:
```sh
npm run watch
```
